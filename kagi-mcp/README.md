# kagi-mcp

MCP server exposing [Kagi Search](https://kagi.com/api) via the Model Context
Protocol. Provides **search** and **extract** tools for LLM agents.

```bash
cargo install --git https://github.com/Silvenga/kagi-mcp.git --locked
```

Full documentation is available in the [workspace README](https://github.com/Silvenga/kagi-mcp).
